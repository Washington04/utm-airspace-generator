# utm-orchestrator
Intended to be a fully functional, UTM simulation that will define a geographical area (Seattle Metro), generate operational intents and constraints, then deconflict those intents using industry consensus prioritization. This project should comply with as may governing regulations as possible, ie ASTM F3548, Part 108, AC146-1, U-space guidance, etc. This project should mimic an ADSP / UAS Service Supplier.
